**Description:**  
A feature-rich mobile application built to modernize the Beauty911 service experience. Users can browse services, schedule appointments, manage profiles, and track bookings in real time.

**Features:**  
- Browse salon services and pricing  
- Schedule, update, and cancel appointments  
- User profile management  
- Real-time booking updates  

**Tech Stack:**  
- Android Studio (Java/Kotlin) or Flutter (Dart)  
- Firebase for backend and real-time database  
- GitHub for version control  

**Installation:**  
1. Clone the repository: `git clone <repo-url>`  
2. Open the project in Android Studio or Flutter IDE  
3. Configure Firebase credentials  
4. Build and run on an Android/iOS device  

**Usage:**  
- Register/login as a user  
- Browse services and make bookings  
- Manage appointments and track updates
